# 📊 Retail Business Intelligence Dashboard

An end-to-end **Data Analytics** project that analyzes retail store sales data to uncover customer behavior, evaluate business performance, segment customers using Machine Learning, forecast future revenue, and provide actionable business insights through an interactive Streamlit dashboard.

---

# 🚀 Project Overview

This project demonstrates the complete data analytics workflow, including:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Business Intelligence Dashboard
- Customer Segmentation using K-Means Clustering
- Cluster Validation using the Elbow Method
- Revenue Forecasting using Linear Regression
- Executive Business Insights
- Interactive Data Explorer

The dashboard enables business users to monitor sales performance, analyze customer behavior, and support data-driven decision-making.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Abdullah-Javed-01/retail-store-sales-analysis
```

Move into the project

```bash
cd retail-store-sales-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
streamlit run app.py
```
